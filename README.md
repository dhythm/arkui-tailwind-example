# Ark UI + Tailwind + React on Vite

## Setting up

### Tailwind

```sh
npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p
```

### Eslint

```sh
npm i -D eslint-plugin-unused-imports@3
```

### Storybook

```sh
npm install @storybook/builder-vite --save-dev
```
